DEMONSTRACIJSKA FORMA (FORMVOZNIPARK)
───────────────────────────────────────────────────────────────────────────

A) PREGLED VOZIL (Levo stran)
   ├─ ListBox s svemi vozili v voznem parku
   ├─ Format: "Marka Model (RegistarskaBroj)"
   └─ Klik na vozilo → prikaže podrobnosti

B) PODROBNOSTI VOZILA
   ├─ Marka in model
   ├─ Registrska tablica
   ├─ Masa (kg)
   ├─ Število vrat
   ├─ Motor (cc)
   └─ Konjska moč (HP)

C) STATISTIKA VOZNEGA PARKA
   ├─ Skupaj vozil: N
   ├─ Povprečna masa: XXX kg
   ├─ Najtežje vozilo: Marka Model (XXX kg)
   └─ Najlakše vozilo: Marka Model (XXX kg)

D) UPRAVLJANJE VOZIL

   ├─ DODAJ VOZILO
   │  ├─ Vnesite marko
   │  ├─ Vnesite registarsko tablico (min. 3 znaki)
   │  ├─ Vnesite maso (kg)
   │  ├─ Izberite barvo
   │  ├─ Vnesite število vrat (2, 4 ali 5)
   │  ├─ Vnesite zapreminę motora (cc)
   │  └─ Pritisnite "Dodaj vozilo"
   │
   ├─ OBRIŠI VOZILO
   │  ├─ Izberite vozilo iz seznama
   │  └─ Pritisnite "Obriši vozilo"
   │
   └─ DEMONSTRACIJA OPERATORJEV
      ├─ Pritisnite "Demo preopterečenja operatorjev"
      ├─ Prikazuje poređenje prve dve vozili
      └─ Prikazuje rezultate: >, <, ==, *

DEMONSTRACIJA OOP PRINCIPOV - PRAKTIČNO
───────────────────────────────────────────────────────────────────────────

1. KAPSLACIJA
   └─ Registrska tablica je zasebna (private)
   └─ Dostopamo skozi javno svojstvo
   └─ Brez direktnega dostopa do internalne strukture

2. KONSTRUKTORJI
   └─ Vsako novo vozilo povečuje interno brojač
   └─ Inizijalizacija je obvezna preko konstruktorja

3. SVOJSTVA
   └─ BrojVrata: Samo 2, 4 ali 5 (validacija)
   └─ Masa: readonly - ne morete spremeniti po kreiranju

4. STATIČNE METODE
   └─ VozniParkManager.DodajVozilo() - brez instance
   └─ VozniParkManager.DajSvaVozila() - vrne seznam

5. DEDOVANJE
   └─ AutoVozilo nasleduje Model
   └─ Pristopate lastnostim iz cele hierarhije

6. OPERATORJI
   └─ Prvo vozilo > Drugo vozilo (primerjava mas)
   └─ Prvo vozilo * 1.2 (povečanje mase)

POGOSTO ZASTAVLJENA VPRAŠANJA (FAQ)
───────────────────────────────────────────────────────────────────────────

V1: Kaj se zgodi, če hočem postaviti BrojVrata = 3?
A: Sistem to ne dopušči! Svojstvo ima validacijo - samo 2, 4 ali 5.

V2: Kako se maž povečuje, ko dodam vozilo?
A: Preko konstruktorja - svaki novo vozilo kliče _brojVozila++

V3: Kaj je ReadOnly svojstvo?
A: GodisnjaMasa je readonly - postavite ga samo v konstruktorju.

V4: Kako deluje operator *?
A: AutoVozilo * 1.2 vrne novo vozilo z povečano maso za 20%.

V5: Kaj je dedovanje?
A: AutoVozilo nasleduje Model - ima dostop do vseh njegovih lastnosti.
