ANALIZA KODA FAZE 2
═════════════════════════════════════════════════════════════════════════════

POLIMORFIZEM:
  Vozilo.Opis() - virtuelna metoda
    ├─ Znamka.Opis() - override
    │  └─ "Znamka: {ImeZnamke}, Tip: {Tip}"
    ├─ Kategorija.Opis() - override
    │  └─ "Kategorija: {ImeKategorije}, Znamka: {ImeZnamke}..."
    ├─ Model.Opis() - override
    │  └─ "Model: {ImeModela} | Znamka: {ImeZnamke}..."
    └─ AutoVozilo.Opis() - override
       └─ "Auto: {ImeZnamke} {ImeModela} ({RegistarskiBroj})..."

ABSTRAKTNE KLASE:
  AutoPart (abstract)
    ├─ abstract int Weight { get; }
    ├─ abstract int AffectHP(Model m);
    └─ virtual string GetDescription()
       ├─ EnginePart - implementacija sa weight=150, boost=_boost
       ├─ SuspensionPart - implementacija sa weight=200, AffectHP=0
       └─ BrakePart - implementacija sa weight=100, AffectHP=-10

VMESNIKI:
  IPerformance
    └─ int GetPerformanceBoost()
       └─ Implementiran u Model

  IVoziloComparable : IComparable<Vozilo>
    └─ int CompareTo(Vozilo other)
       └─ Implementiran u Vozilo sa override-om

  IVoziloKolekcija
    ├─ int Count { get; }
    ├─ Vozilo this[int index]
    ├─ Add(Vozilo vozilo)
    └─ Remove(Vozilo vozilo)
       └─ Implementiran u AutoVozilo

  IEquatable<Model>
    └─ bool Equals(Model other)
       └─ Implementiran u Model

INDEKSATORI:
  Model[int] - pristup delovima
    ├─ Get: vraća AutoPart sa danim indeksom
    └─ Set: postavlja novi AutoPart

  Model[string] - pristup svojstvima
    ├─ Get: vraća svojstvo kao string
    └─ Podrška: "imemodela", "imeznamke", "hp", "tip"

  AutoVozilo[int] - pristup vozilima
    ├─ Get: vraća Vozilo sa danim indeksom
    └─ Set: postavlja novo Vozilo
