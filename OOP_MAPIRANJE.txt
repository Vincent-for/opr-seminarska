6. PREOPTEREĆENJE OPERATORA (Operator Overloading)
───────────────────────────────────────────────────────────────────────────
Lokacija: Vozila.cs - AutoVozilo klasa i Model klasa

AutoVozilo Operatori:
• operator > - poređenje po masi (a1.GodisnjaMasa > a2.GodisnjaMasa)
• operator < - poređenje po masi (a1.GodisnjaMasa < a2.GodisnjaMasa)
• operator == - poređenje po registarskom broju (RegistarskiBroj)
• operator != - negacija == operatora
• operator * - multiplikacija mase sa faktorom
  Primer: auto * 1.2 → novo vozilo sa masom povećanom za 20%

Model Operatori:
• operator > - poređenje po HP (a.HP > b.HP)
• operator < - poređenje po HP (a.HP < b.HP)
• operator == - poređenje po modelskim svojstvima
• operator != - negacija == operatora

7. DEDOVANJE (Inheritance)
───────────────────────────────────────────────────────────────────────────
Lokacija: Vozila.cs

Hijerarhija Klasa:
────────────────────────────────────────
    Vozilo (Abstract - bazna klasa)
      │
      └─→ Znamka : Vozilo
           │
           └─→ Kategorija : Znamka
                │
                └─→ Model : Kategorija, IEquatable<Model>, IPerformance
                     │
                     └─→ AutoVozilo : Model

Dedovanje u Praksi:
• AutoVozilo nasledjuje Model sa: class AutoVozilo : Model
• Poziva baznu klasu sa: base(tip, imeZnamke, imeKategorije, imeModela, hp)
• Pristupa baznim svojstvima: ImeZnamke, ImeModela, HP (iz Model klase)
• Overriduje metodu (ako postojala): ToString()
