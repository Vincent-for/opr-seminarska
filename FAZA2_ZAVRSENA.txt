ŠTO JE NOVO U FAZI 2
───────────────────────────────────────────────────────────────────────────

1. POLIMORFIZEM
   └─ Virtuelne metode u Vozilo baznoj klasi
   └─ Override metode u Znamka, Kategorija, Model, AutoVozilo
   └─ Različiti odgovori za istu metodu po tipu klase
   └─ Demonstracija kroz Opis(), CompareTo(), PrikaziKarakteristike()

2. ABSTRAKTNI RAZREDI
   └─ AutoPart - abstraktna klasa (ne može se direktno instancirati)
   └─ Abstrakt svojstva: Weight
   └─ Abstrakt metode: AffectHP()
   └─ 3 konkretne implementacije:
      ├─ EnginePart (povećava HP za +50)
      ├─ SuspensionPart (povećava grip za +15%)
      └─ BrakePart (smanjuje HP za -10)

3. VMESNIKI
   └─ IPerformance - interfejs za performanse
      ├─ Implementiran u: Model, AutoVozilo
      └─ GetPerformanceBoost() - HP + težina delova

   └─ IVoziloComparable - interfejs za poređenje
      ├─ Implementiran u: Vozilo
      └─ CompareTo() - poređenje vozila

   └─ IVoziloKolekcija - interfejs za upravljanje kolekcijom
      ├─ Implementiran u: AutoVozilo
      ├─ Count { get; }
      ├─ Add(), Remove()
      └─ this[int index]

   └─ IEquatable<Model> - interfejs za jednakost
      ├─ Implementiran u: Model
      └─ Equals(Model other)

4. INDEKSATORJI
   └─ Model[int] - pristup delovima po broju
      ├─ Get: vraća AutoPart sa danim indeksom
      └─ Set: postavlja novi AutoPart

   └─ Model[string] - pristup svojstvima po imenu
      ├─ Podrška: "imemodela", "imeznamke", "hp", "tip"
      └─ Vraća vrednost kao string

   └─ AutoVozilo[int] - pristup vozilima iz kolekcije
      ├─ Get: vraća Vozilo sa danim indeksom
      └─ Set: postavlja novo Vozilo

POLIMORFNA PRIMENA - PRAKTIČNI PRIMERI
───────────────────────────────────────────────────────────────────────────

Primer 1: Različiti Opis() za svaki tip
```csharp
Vozilo v1 = new Znamka("Avto", "BMW");
Vozilo v2 = new Model(..., "BMW", "Sport", "M3", 473);
Vozilo v3 = new AutoVozilo(..., "NIS-100-XX", 4, 1200, 1500);

// POLIMORFIZEM - ista metoda, različiti odgovori!
v1.Opis();  // "Znamka: BMW, Tip: Avto"
v2.Opis();  // "Model: BMW M3 | Znamka: BMW | Kategorija: Sport | HP: 473"
v3.Opis();  // "Auto: BMW M3 (NIS-100-XX)\nMasa: 1500kg | Motor: 1200cc..."
```

Primer 2: Virtuelne metode - CompareTo()
```csharp
Vozilo a = new Znamka("Avto", "BMW");
Vozilo b = new Znamka("Avto", "Audi");
int rezultat1 = a.CompareTo(b);  // Poređenje po ImeZnamke

Vozilo m1 = new Model(..., "BMW", "M3", 473);
Vozilo m2 = new Model(..., "BMW", "X5", 335);
int rezultat2 = m1.CompareTo(m2);  // Poređenje po HP

Vozilo auto1 = new AutoVozilo(..., 1500);
Vozilo auto2 = new AutoVozilo(..., 2000);
int rezultat3 = auto1.CompareTo(auto2);  // Poređenje po masi
```

Primer 3: Abstraktne klase - AutoPart
```csharp
// GREŠKA - ne može se direktno instancirati
// AutoPart part = new AutoPart();  // CS0144 - abstract class

// OK - samo kroz konkretne klase
AutoPart engine = new EnginePart(50);
AutoPart suspension = new SuspensionPart(15);
AutoPart brake = new BrakePart(90);

// Polimorfna primena
List<AutoPart> parts = new List<AutoPart> { engine, suspension, brake };
foreach (var part in parts)
{
    // GetDescription() je virtuelna - različiti izlazi!
    Console.WriteLine(part.GetDescription());
    // Engine Part - Boost: +50 HP...
    // Suspension Part - Grip: +15%...
    // Brake Part - Power: 90%...
}
```

Primer 4: Vmesniki - IVoziloKolekcija
```csharp
AutoVozilo auto = new AutoVozilo(...);

// Interfejs IVoziloKolekcija
auto.Add(new Vozilo("Avto"));
auto.Add(new Znamka("Avto", "BMW"));
auto.Add(new Model(...));

Console.WriteLine(auto.Count);  // 3

Vozilo v = auto[0];  // Indeksator!
```

Primer 5: Indeksatori
```csharp
Model model = new Model(...);
model.ApplyPart(new EnginePart(50));

// Indeksator po broju
AutoPart del = model[0];
Console.WriteLine(del.GetDescription());

// Indeksator po stringfu
string imeModela = model["imemodela"];
string hp = model["hp"];

// AutoVozilo indeksator
AutoVozilo auto = new AutoVozilo(...);
auto.Add(new Vozilo("Avto"));
Vozilo v = auto[0];
```
